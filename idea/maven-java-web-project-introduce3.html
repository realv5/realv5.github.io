项目

  • 项目地址：https://github.com/judasn/spring-boot-showcase

开发环境

  • JDK 8（理论上支持 JDK 6、JDK 7、JDK 8）

  • Maven 3.3.9
  • Tomcat 8（Spring Boot 内置）
  • IntelliJ IDEA 2017.2.1
  • 所有编码：UTF-8
